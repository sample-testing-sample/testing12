NeuroTract: Brain White Matter Analysis

NeuroTract processes diffusion MRI data to analyze brain white matter connectivity. The platform provides:

Processing Pipeline

  • Stage 1: Data Ingestion - Validates NIfTI files and gradient tables
  • Stage 2: Preprocessing - Motion correction, brain extraction, bias field correction
  • Stage 3: Microstructure Modeling - DTI and CSD analysis
  • Stage 4: Tractography - Probabilistic fiber tracking
  • Stage 5: Surface Generation - Brain mesh and parcellation mapping
  • Stage 6: Connectome Construction - Graph representation of brain connectivity
  • Stage 7: Graph Analysis - Network metrics and statistics

Input Requirements

  • DWI File: Diffusion-weighted NIfTI image (.nii.gz)
  • B-values: Text file with diffusion weighting strengths
  • B-vectors: Text file with gradient directions
  • Optional: T1-weighted anatomical scan, brain mask, parcellation atlas

Output Metrics

  • Fractional Anisotropy (FA) - Directional diffusion measure
  • Mean Diffusivity (MD) - Average water diffusion
  • Connectivity Matrix - Brain region connectivity strengths
  • Graph Metrics - Network properties (clustering, modularity, centrality)

Getting Started

To use NeuroTract:

  1. Prepare your MRI data in BIDS format or provide file paths directly
  2. Click "Open NeuroTract Interface" to access the web platform
  3. Upload your data and configure processing parameters
  4. Monitor processing progress in real-time
  5. Review results with interactive 3D visualizations
  6. Export metrics for clinical interpretation
